# Sales Forecasting Dashboard – Power BI

## Overview
This project is a sales forecasting dashboard built using Power BI. The dashboard analyzes historical sales data and provides insights into future sales trends to support business decision-making.

## Features
- Interactive Power BI dashboard
- Sales trend analysis
- KPI tracking
- Forecasting visuals
- Regional and category performance analysis
- Dynamic filters and slicers

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel

## Key Skills Demonstrated
- Data Visualization
- Business Intelligence
- Data Cleaning
- Data Modeling
- Dashboard Development
- DAX Calculations
